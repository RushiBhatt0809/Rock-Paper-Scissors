# 🪨 Rock Paper Scissors

A Rock Paper Scissors game built with React.js

## ✨ Features
- Play against the computer
- Real-time score tracking (Wins, Losses, Ties)
- Responsive mobile-width layout
- Ocean Dark theme

## 🛠️ Built With
- React.js
- Vite
- CSS3

## 🚀 Run Locally
git clone https://github.com/yourname/rock-paper-scissors.git

cd rock-paper-scissors

npm install

npm run dev
